# ⏳ Countdown Timer

A simple countdown timer built using Python.  
It counts down from a given number to zero with a 1-second delay.

## 🚀 Features
- Takes user input for countdown time
- Displays countdown step by step
- 1-second delay between numbers
- Prints celebration message at the end

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Open terminal or command prompt
2. Navigate to project folder
3. Run the file:
   python countdown.py

## 📌 How It Works
- Uses a loop to count backwards
- Uses time.sleep(1) for 1-second delay
- Prints numbers until it reaches zero

## 💡 Example
Input: 5  
Output:
5  
4  
3  
2  
1  
🎉 WOHOOO! Happy New Year
